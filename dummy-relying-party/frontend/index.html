Relying party

This is a relying party that can request credentials from any issuer.

You must provide all the data necessary to request the credential.

Starting by logging in with the same Internet Identity used by the issuer.

Enter origin of the issuer to request the credential.

Enter the credential type you'd like to request, eg: "VerifiedAge".

Enter the arguments of the credential, eg: "minAge: 18".
The arguments are added to the credential spec below.
